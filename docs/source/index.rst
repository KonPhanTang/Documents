Welcome to Bitcoin Developer Documents (TH)
===================================
_เรียนรู้ Bitcoin และเริ่มสร้างแอปพลิเคชันที่ใช้ Bitcoin_

**เริ่มต้น**

ไซต์นี้มีจุดมุ่งหมายเพื่อให้ข้อมูลที่คุณต้องการเพื่อทำความเข้าใจ Bitcoin ในภาษาไทย และ เริ่มสร้างแอปพลิเคชันที่ใช้ Bitcoin หากต้องการใช้เอกสารนี้ ให้เกิดประโยชน์สูงสุด [ตรวจสอบให้แน่ใจว่าคุณกำลังใช้งาน node](https://bitcoin.org/en/full-node#what-is-a-full-node)

สำหรับการสนับสนุนทางเทคนิค เราขอแนะนำ Bitcoin Stack Exchange สำหรับข้อผิดพลาดหรือข้อเสนอแนะที่เกี่ยวข้องกับเอกสารนี้ โปรดเปิดปัญหาใน GitHub

**รับทราบ**

เอกสารนี้จะเป็นไปไม่ได้หากไม่มีการมีส่วนร่วมมากมายในโครงการ Bitcoin ในช่วงหลายปีที่ผ่านมาจากนักพัฒนาหลักและคนอื่นๆ อย่างไรก็ตาม ขอขอบคุณเป็นพิเศษสำหรับ David Harding ซึ่งในปี 2014 ได้ช่วยนำความพยายามในการเรียบเรียงและรวบรวมเนื้อหาจำนวนมากที่พบในที่นี้ นอกจากนี้ สำหรับ Cornelius Schumacher สำหรับการคิดหาวิธีใหม่ในการขยายเอกสารสำหรับนักพัฒนาที่นำไปสู่ไซต์นี้
